Process Scheduling Algorithms

I worked on recreating several core process scheduling algorithms, including Shortest Remaining Time (SRT), Shortest Job First (SJF), and First Come First Serve (FCFS). The development of the SJF algorithm was done using Python, providing an efficient solution for scheduling processes based on job duration. These algorithms are crucial for optimizing CPU utilization and minimizing process waiting times in operating systems.

Technologies used: Python.
