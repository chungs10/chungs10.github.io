Ken Thompson Hack Recreation

As part of this project, my team recreated a hack inspired by the Ken Thompson hack, where a backdoor was injected into a system's login files via the compiler. This was done using C, and involved researching how to modify the compiler to achieve this effect. The project allowed us to better understand the mechanics of compiler-level vulnerabilities and demonstrated how subtle backdoors can be introduced into secure systems without detection.

Technologies used: C programming language, compiler modification.
