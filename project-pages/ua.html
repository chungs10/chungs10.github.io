United Airlines Cargo Weight and Balance Evaluation

In collaboration with three team members, I contributed to the development of a web application for United Airlines designed to evaluate the weight and balance of an aircraft's cargo hold. My primary role involved designing and developing the frontend using HTML, CSS, and JavaScript to create a user-friendly interface. Additionally, I contributed to the backend using Python, ensuring the seamless processing of cargo data. This project combined several technologies, including HTML, CSS, JavaScript, and Python, to deliver an efficient and secure solution.
