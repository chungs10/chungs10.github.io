Food Stock Management System

My group developed a web application for managing food stock quantities, specifically focusing on initiating alerts for the donation of food items nearing their best-before dates. This project aimed to ensure that surplus food would be donated to underprivileged groups before spoilage. My primary role was developing the donation page and contributing to the login system using PHP. The system was built using HTML, CSS, JavaScript, and PHP, ensuring functionality for both front-end and back-end operations.

Technologies used: PHP, HTML, CSS, JavaScript.
