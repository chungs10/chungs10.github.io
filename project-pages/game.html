Introduction to IT Project: Simulation Game of Crashing into Mars

This project was part of an introductory IT course, where we were tasked with applying our basic web development skills to build an interactive web-based game. My group of two decided to create a simulation game that challenges the player to survive a crash landing on Mars.

The gameplay involves managing resources and making decisions to improve the chances of survival in the harsh Martian environment. The game was developed using HTML, CSS, and JavaScript, combining front-end design and basic game logic to create a simple but engaging user experience.

During development, we focused on building a visually appealing user interface using CSS, while JavaScript was used to handle the game mechanics such as tracking resources and determining outcomes based on player choices. The project served as a great learning experience in applying web development skills.
